A new logo for nerds

At Q42, technology is a lens for creativity. So our logo itself is also technology: a code logo.

How to use

  1. Add <script src="http://logo.q42.nl/logo.js"></script> to the head of your page.
  2. Place a logo using <q42logo>.
  3. Use CSS to style & position.

Optionally, you can add attributes to <q42logo>

  • data-type="svg|canvas|webgl"
    (default = canvas)
  • data-theme="green|black|white"
    (default = green)

The Q42 logo is all bezier curves. The control points and coordinates are in the blueprint below for you to recreate for whatever purpose.

Even more old–fashioned

Rumors exist that outside of the modern world, some still use paper to get stuff done. We don't know why or what for. But if someone asks for something called a 'vector file', just give them this EPS file.

Physical prototyping

  • To carve it in wood or metal on our Carvey, fire up Easel and import this SVG file.
  • To work it into a 3D printed model, import a DXF, STL, or SVG file into your modeler.

Contribute (Q42-ers only)

For all your awesome ideas, like doodles on specific days, attract animations, hidden minigames, whatever:

  1. Clone from Github.
  2. Create a plugin to add behavior, or just hack away.
  3. Make a pull request.
  4. Note that any changes will be visible on all interactive Q42 logos everywhere.
